博客搭建
  1. 前置条件
  2. 第一步: 安装 hexo 框架并生成博客
  3. 第二步：配置 A4 主题
  4. 第三步：关联 github

前置条件

  • 具有 Web 前端初级技能水平
  • 已配置好的 Node 环境
  • 已拥有 Github 账号,并熟悉 GIT 操作
  • 熟悉 Markdown 文档语法

第一步: 安装 hexo 框架并生成博客

  • 使用 npm 安装 hexo 框架： npm install hexo-cli -g

  • 到文件目录下使用 hexo 初始化博客：hexo init 你的博客文件名

  • 进入生成好的博客：cd 你的博客文件名

  • 安装博客依赖：npm install

  • 确认博客是否可以运行：hexo server （博客默认运行在本地 4000 端口）

  • 恭喜~到这一步，博客已成功运行

  • 如需深入了解 hexo 框架,请点击这里

第二步：配置 A4 主题

  • 在博客根目录下安装主题：npm install hexo-theme-a4@latest

  • 在博客根目录创建_config.a4.yml 文件,将以下内容复制进入文件（对主题的所有个性化配置都将在_config.a4.yml 文件中进行，可按照文件中的注释自行配置）

  • 在博客根目录选中_config.yml 将主题切换为 A4(全局搜索 theme 字段，将 a4 填入 theme 后)

  • 恭喜~到这一步，主题已经成功安装

  • 请点击这里深入了解 A4 主题以及博客内容的编写

第三步：关联 github

  • 在 github 中创建以自己 账户名称.github.io 为文件名的项目，并将权限设为开放 Public
  • 在博客根目录选中_config.yml 配置 deploy(全局搜索 deploy 字段,将以下内容修改为对应数据)
  • 在博客根目录运行 hexo clean 清理生成的资源
  • 在博客根目录运行 hexo generate 生成的资源
  • 在博客根目录运行 hexo deploy 推送资源到 gihub（请确保本地已有 github 凭据）
  • 等待 5 分钟，访问 https://github.com/github账户名称/github账户名称.github.io.git 即可
  • 恭喜~到这一步，已成功部署到 github
  • 如需深入了解推送相关,请点击这里
